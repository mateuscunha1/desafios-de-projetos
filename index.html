História da Web

A história da web começa com o surgimento dos primeiros computadores. A primeira máquina eletrônica programável foi criada em 1937 por John Atanasoff e Clifford Berry. Em 1941, Konrad Zuse criou o primeiro computador eletromecânico. Em 1945, John von Neumann propôs a arquitetura de um computador que se tornou a base para a maioria dos computadores modernos.

A World Wide Web (WWW) foi inventada no CERN (Centro Europeu para Física Nuclear) por Tim Berners-Lee no período de 1989 a 1991. Berners-Lee e seus companheiros desenvolveram versões iniciais de HTML, HTTP, um servidor web e um Browser. O primeiro servidor web e o primeiro navegador web gráfico foram criados em 1990 por Tim Berners-Lee enquanto trabalhava no CERN. Ele batizou sua nova janela para a internet de "WorldWideWeb".

Grandes nomes que contribuíram para o desenvolvimento da web incluem Tim Berners-Lee, Robert Cailliau, Marc Andreessen, Eric Bina e muitos outros.


O que são Clients

Clients são entidades que solicitam e recebem serviços de outras entidades, chamadas de servers. No contexto da web, os clients são geralmente os navegadores, as aplicações web e os dispositivos móveis que acessam conteúdos e funcionalidades oferecidos pelos servers.

Os navegadores são programas de software que permitem aos usuários visualizar e interagir com páginas web. Eles são responsáveis por enviar requisições HTTP aos servers e renderizar as respostas em HTML, CSS e JavaScript. Alguns exemplos de navegadores são Google Chrome, Mozilla Firefox, Microsoft Edge e Safari.

As aplicações web são programas que rodam dentro dos navegadores e oferecem funcionalidades específicas aos usuários. Elas podem ser acessadas por meio de URLs (Uniform Resource Locators) e não precisam ser instaladas no computador ou no dispositivo móvel. Alguns exemplos de aplicações web são Gmail, Facebook, YouTube e Twitter.

Os dispositivos móveis são aparelhos portáteis que permitem aos usuários acessar a internet por meio de redes sem fio. Eles podem ter navegadores integrados ou aplicativos dedicados para acessar determinados serviços web. Alguns exemplos de dispositivos móveis são smartphones, tablets, smartwatches e laptops.


O Que são Servers

Servers, ou servidores, são computadores que fornecem serviços ou recursos para outros computadores, chamados de clientes, em uma rede. Os servidores podem ter diferentes funções, dependendo do tipo de serviço que oferecem. Por exemplo, um servidor web hospeda sites e permite que os clientes acessem esses sites através de um navegador. Um servidor de e-mail envia e recebe mensagens eletrônicas dos clientes. Um servidor de arquivos armazena e compartilha arquivos entre os clientes.

Os servidores precisam de hardware e software específicos para desempenhar suas tarefas. O hardware de um servidor geralmente é mais potente e robusto do que o de um computador comum, pois precisa lidar com uma grande demanda de solicitações dos clientes. O hardware de um servidor pode incluir processadores, memórias, discos rígidos, placas de rede, fontes de alimentação e sistemas de refrigeração. O software de um servidor pode incluir sistemas operacionais, aplicativos, bancos de dados, protocolos e ferramentas de segurança.

Existem diferentes tipos de servidores, que podem ser classificados de acordo com o tipo de serviço que oferecem, o tipo de hardware que utilizam ou o tipo de hospedagem que possuem.


Linguagens de Programação

Linguagens de programação são sistemas de símbolos e regras que permitem criar instruções para que um computador execute determinadas tarefas. Existem diferentes tipos de linguagens de programação, cada uma com suas vantagens e desvantagens, e que podem ser usadas para diferentes fins.

Uma forma de classificar as linguagens de programação é pela forma como elas interagem com o navegador web, que é o programa que permite acessar páginas e aplicativos na internet. Nesse sentido, podemos falar de linguagens server-side e client-side.

As linguagens server-side são aquelas que rodam no servidor, ou seja, no computador que hospeda o site ou o aplicativo. Elas são responsáveis por gerenciar os dados, a lógica e a segurança da aplicação. Alguns exemplos de linguagens server-side são PHP, Python, Ruby e Java.

As linguagens client-side são aquelas que rodam no navegador do usuário, ou seja, no computador que acessa o site ou o aplicativo. Elas são responsáveis por criar a interface gráfica, os efeitos visuais e a interatividade da aplicação. Alguns exemplos de linguagens client-side são JavaScript, CSS e HTML.

HTML é uma sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. Ela é usada para definir a estrutura e o conteúdo das páginas web, usando tags que indicam os elementos como títulos, parágrafos, imagens, links, etc.


HTML na Prática

  1. Estrutura básica do HTML

    A estrutura básica do html é formada por três partes principais: o doctype, o head e o body. O doctype declara o tipo de documento html que estamos usando, e deve ser a primeira linha do código. O head contém informações sobre a página, como o título, os metadados, os estilos e os scripts. O body contém o conteúdo visível da página, como textos, imagens, links, tabelas, formulários etc.


  2. Tags

    O HTML é composto por elementos chamados de tags, que são escritos entre sinais de menor e maior (< e >). As tags definem o conteúdo e o layout da página, e podem ter atributos que especificam propriedades adicionais. Cada tag tem uma tag de abertura e uma tag de fechamento, que indicam onde o elemento começa e termina. Por exemplo, a tag < p > cria um parágrafo, e deve ser fechada com < /p >. Algumas tags são auto-fechadas, ou seja, não precisam de uma tag de fechamento. Por exemplo, a tag < img > insere uma imagem na página, e pode ter atributos como src (fonte da imagem) e alt (texto alternativo).


  3. Atributos

    Os atributos HTML são palavras especiais que modificam o funcionamento ou a aparência de um elemento HTML. Eles são usados dentro das tags de abertura dos elementos, separados por espaços. Cada atributo tem um nome e um valor. Alguns atributos são globais, ou seja, podem ser usados em qualquer elemento, como o atributo class, que define uma classe CSS para o elemento. Outros atributos são específicos para certos elementos, como o atributo src, que define a URL de uma imagem ou de um script. Os atributos HTML podem melhorar a acessibilidade, a semântica e a funcionalidade da página web.


  4. Listas

    As listas HTML são elementos que permitem organizar e exibir informações de forma estruturada e hierárquica. Existem três tipos principais de listas HTML: ordenadas, não ordenadas e de definição. As listas ordenadas usam a tag < ol > e mostram os itens em uma sequência numérica, alfabética ou romana. As listas não ordenadas usam a tag < ul > e mostram os itens com símbolos como círculos, quadrados ou discos. Em todos os tipos de listas, cada item é representado pela tag < li >, que significa lista de itens.
